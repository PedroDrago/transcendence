# Project Context

- This project is a social network.
- It is not a game project.
- Avoid game-themed vocabulary in code, docs, UI text, fallback names, and examples unless the user explicitly asks for it.
- Prefer generic social-network terminology such as `user`, `profile`, `post`, `feed`, `follow`, `message`, and `account`.
- When committing in this repo, group relevant code chunks into separate commits instead of collapsing everything into one giant commit.
- Use concise conventional commit messages.
- Keep PR descriptions concise as well.
